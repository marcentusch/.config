# mo.config
My .config folder

To use this, clone this to home directory, on Mac `cd ~`
Then rename the directory to .config

## Aliases
alias ls="exa -la" \
alias vim="nvim" \
alias ..="cd .." 

## Tools
- Kitty: Terminal emulator
- NVIM: Text editor
- Starship: shell prompt
- gitui: TUI git client
